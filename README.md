# PDF splitter

## Split a .pdf file and save each individual page as PDF

## Instructions

- Drop the pdf to be splitted in the `DROP` folder
- Double Click on the `main.bat` file to run program
- Splitted pdf files are in the `OUTPUT` folder

### Packages used

- PyPDF2
- time (Inuilt)
- glob (Inbuilt)
- os (Inbuilt)
